# rpi-libm2k-setup

Reproducible Raspberry Pi setup for building and installing:

- libiio
- libm2k

## Intended use

Validated on a Raspberry Pi with Raspberry Pi OS / Debian-based environment.

## Quick start

```bash
git clone https://github.com/AlboSoc/rpi-libm2k-setup.git
cd rpi-libm2k-setup
chmod +x setup-rpi-libm2k.sh scripts/*.sh
./setup-rpi-libm2k.sh
